.. _astrodata_logo:

The Logo
========

The astrodata logo was created by Teal.
    The astrodata logo uses the Python blue and the Gemini yellow for the star.
    The logo is a stylized representation of the letter A, for astrodata,
    and illustrates a dragon hatching from an egg, which is a reference to the
    DRAGONS project from which the astrodata package originated.
